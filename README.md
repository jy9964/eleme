一个 基于 vue.js 的仿饿了么SPA App

项目中用到VueRouter、Axios、BetterScroll库，基于移动端页面开发的SPA应用，对ios及安卓的兼容性良好,对不同分辨率也做了适配。项目采用组件化开发，便于维护。
